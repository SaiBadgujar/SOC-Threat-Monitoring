🛡 **SOC & Threat Monitoring Lab**

📌 **Project Overview**

This repository documents my hands-on Security Operations Center (SOC) lab practice focused on threat detection, log analysis, packet monitoring, and incident classification using industry-relevant tools.
The project simulates real-world security monitoring scenarios including brute force attacks, port scanning, and web-based exploitation attempts.

🎯 **Objectives**

Monitor and analyze simulated security events
Detect suspicious activities using log correlation
Perform packet-level traffic analysis
Classify and document security incidents
Practice SOC incident response workflow

🛠 **Tools Used**

Splunk – Log monitoring & SIEM-style analysis
Wireshark – Network packet inspection
Kali Linux – Attack simulation
Nmap – Port scanning detection
Metasploit – Exploitation simulation

🔍 **Activities Performed**

1️⃣ Log Monitoring & Alert Analysis
Ingested logs into Splunk for monitoring simulated attack activity
Created basic search queries to identify suspicious patterns
Analyzed authentication failures and abnormal login attempts
Correlated events to detect brute force attempts

2️⃣ Network Traffic Analysis
Captured live network traffic using Wireshark
Inspected packets for unusual source/destination behavior
Identified scanning activity and abnormal protocol usage

3️⃣ Threat Detection Scenarios
Brute Force Attack Detection
Port Scanning Identification
Web Exploitation Attempt Monitoring
Suspicious Traffic Pattern Analysis

4️⃣ Incident Classification
Categorized incidents as Low / Medium / High severity
Documented findings in structured incident reports
Followed basic Incident Response lifecycle:
Identification
Analysis
Containment
Documentation

📊 **Key Learning Outcomes**

Practical understanding of SOC monitoring workflows
Experience working with SIEM-style log analysis
Improved ability to detect anomalies through log correlation
Hands-on exposure to packet-level traffic investigation
Strengthened incident documentation and reporting skills
